| **REST API**                                                         |                                                                                                                                                                                                                                 |
| Интеграционное API добавления пользователей в группу                 | Реализовано REST API управления группами: создание, редактирование, удаление групп, добавление и удаление пользователей, получение списка групп                                                                                 |
| Повторный вызов WebRTC-участника в мероприятие                       | Реализовано REST API повторного вызова WebRTC-участника в мероприятие (вызов доступен участникам с правом приглашения)                                                                                                          |
| Очистка остаточной информации из оперативной памяти                  | Добавлена функция автоматической очистки остаточной информации (паролей пользователей) из оперативной памяти                                                                                                                    |
| Расширенные статусы пользователей                                    | Реализовано REST API управления текстовым статусом присутствия пользователя в системе (например, "На совещании", "В отпуске" и т.д.)                                                                                            |
| Возможность изменения гостем своего имени                            | Реализовано REST API изменения незарегистрированным пользователем (гостем) собственного имени в мероприятии                                                                                                                     |
| **Внутренние модули**                                                |                                                                                                                                                                                                                                 |
| Поддержка внешних файловых хранилищ на основе GlusterFS              | Добавлена поддержка внешних файловых хранилищ на основе GlusterFS (адрес хранилища должен начинаться с glusterfs://)                                                                                                            |
| Conversion:                                                          | Обновлены система управления конфигурацией Zookeper и сетевой фреймворк Netty до последних версий                                                                                                                               |
