## Сервер 20.0

### Администрирование

#### Поддержка больших RTMP-вебинаров

Поддержка проведения системой IVA MCU масштабных вебинаров численностью до 10 000 участников, которое реализуется распределением трансляции FLV-вебинара между пользователями через различные доступные сервера Nginx-media

#### Бронирование ресурсов подписки

Реализован функционал бронирования ресурсов подписки, используемой в мероприятии (звонке, комнате) на определенный период времени.

Ограничение на Максимальное количество WebRTC- участников / VVoIP-участников мероприятий являются бронируемыми ресурсами подписки

### REST API

#### Управление пользовательскими шаблонами мероприятий

Реализовано клиентское REST API управления пользовательскими шаблонами мероприятий

#### Редактирование периодического мероприятия

Реализовано клиентское REST API редактирования всего периодического мероприятия

#### Интеграционное API для ВКурсе

Реализовано интеграционное REST API управления подписками пользователей

#### Клавиатурные действия в бот API

Изменение функции отправки сообщения в бот API и объектов чата в клиентском REST API

#### Мягкий мьют участника мероприятия

Реализовано клиентское REST API принудительного отключения медиа трансляций (аудио / видео по отдельности) отдельных участников мероприятия (без запрета на вещание отключаемого типа) и отправки запроса отдельным участникам мероприятия на включение медиа трансляции (аудио / видео по отдельности)

#### Мягкий мьют медиатрансляции всем участникам мероприятия

Мягкий мьют медиа трансляции всем участникам мероприятия

### Внутренние модули

#### Monitoring

Возможность безусловного разрешения входа пользователя root

### Исправления

- Monitoring: Ошибка запуска SSH-службы после настройки
- Monitoring: Отсутствие метода builder() в MonitoringDnsConfig
- Monitoring: Ошибка синхронизации resolv.conf
- VVoIP Signalling Gateway: Ошибка обработки запроса авторизации на bye
- VVoIP Signalling Gateway: Некорректное логирование события «marked as reachable»
- VVoIP Signalling Gateway: Ошибка сохранения route list на уровне диалога

## iOS клиент 16.1

### Мероприятия

#### Создание мероприятия

- Добавлена возможность создания запланированных мероприятий и запуска мгновенных мероприятий

#### Редактирование мероприятия

- Приложение поддерживает возможность редактирования разового мероприятия или сессии периодического мероприятия для участников с ролью Организатор или Модератор
- Мягкий мьют участника мероприятия
  Для участника мероприятия добавлены следующие уведомления:
  · Изменение прав на трансляцию основного контента (разрешение / запрет на трансляцию)
  · Выключение трансляции основного контента модератором

· Запрос модератора на включение трансляции основного контента

### Комнаты

#### Создание комнаты

Добавлена возможность создания комнат

#### Редактирование комнаты

Приложение поддерживает возможность редактирования комнаты для участников с ролью Организатор или Модератор

### Чаты

#### Поиск пользователей на сервере

- Добавлена поддержка серверного поиска пользователей при создании чата и добавлении пользователей в чат

### Исправления

- Превышение максимального количества символов при вводе в поле ID мероприятия в окнах подключения к мероприятию по ID
- Дублирование звука при просмотре видеофайла в вебинаре, если настроено распределение вебинара по медиасерверам и выключена настройка "Звук демонстрации видео в потоке демонстрации"
- Некорректный возврат к разделу Чаты при выходе из мероприятия по ссылке SID незарегистрированного пользователя
- Некорректное отображение уведомления о включении камеры при назначении другому гостевому участнику роли докладчика в вебинаре
- Аварийное завершение приложения при назначении докладчику роли гостя в вебинаре
- Некорректный возврат к разделу Чаты при авторизации
- Ошибка подключения к вебинару с паролем, если при входе указать имя гостя из двух и более слов
- Задержка отображения элементов панели управления мероприятием при назначении пользователя докладчиком в большом вебинаре (более 1500 участников)
